# Python Quiz Game

## Overview

The Python Quiz Game is a multiple-choice quiz application developed using Python fundamentals.

The game presents a series of questions, validates user answers, tracks scores, and displays final performance results.

---

## Features

* Multiple-choice questions
* Answer validation
* Score tracking
* Final percentage calculation
* Performance evaluation
* Interactive gameplay
